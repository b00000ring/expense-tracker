## 📌 Описание программы

**Expense Tracker** — это графическое приложение на Python для учёта личных расходов.

Программа позволяет пользователю:

- Добавлять расходы с указанием суммы, категории и даты  
- Удалять выбранные записи  
- Фильтровать расходы по категории  
- Фильтровать расходы по диапазону дат  
- Подсчитывать общую сумму за выбранный период  
- Сохранять данные в формате JSON  
- Загружать данные при повторном запуске  

Приложение предназначено для контроля личных финансов и анализа расходов за определённый период времени.

🖥 Описание интерфейса
1. Блок добавления расхода
Поля:

Сумма (положительное число)
Категория (текст)
Дата (формат YYYY-MM-DD)
Кнопка:

«Добавить»
2. Таблица расходов
Отображает:

Сумму
Категорию
Дату
3. Блок фильтрации
Позволяет:

Фильтровать по категории
Фильтровать по диапазону дат
Кнопки:

«Применить»
«Сбросить»
4. Дополнительные функции
«Удалить выбранный» — удаление записи
«Подсчитать за период» — вывод общей суммы
«Сохранить» — сохранение в JSON
Данные автоматически сохраняются при закрытии программы.

✅ Валидация ввода
Программа проверяет корректность данных:

Сумма:
Должна быть числом
Должна быть больше 0
Дата:
Формат строго YYYY-MM-DD
Категория:
Не может быть пустой
При некорректном вводе выводится сообщение об ошибке.

🧪 Тестирование программы
✅ Позитивные тесты
Ввод:
Сумма: 150
Категория: еда
Дата: 2026-04-25
Ожидаемый результат:

Запись добавляется в таблицу
Фильтрация:
Категория: еда
Ожидаемый результат:

Отображаются только расходы категории «еда»
Подсчёт за период:

По дату: 2026-04-30
Ожидаемый результат:

❌ Негативные тесты

Ожидаемый результат: сообщение об ошибке

Ожидаемый результат: сообщение об ошибке

⚠ Граничные случаи
Сумма = 0 → ошибка
Очень большое число → корректно обрабатывается
Пустой JSON файл → программа не падает
